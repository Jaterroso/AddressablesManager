### Install via Git URL

Open *Packages/manifest.json* with your favorite text editor. Add the following line to the dependencies block.

    {
        "dependencies": {
            "com.addressablesmanager.core": "https://github.com/SujanDuttaMishra/AddressablesManager.git"
        }
    }
